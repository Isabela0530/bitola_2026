# bitola_2026

## Descrição
Aplicativo desenvolvido em Flutter para cálculo da bitola de fios elétricos. O usuário informa a corrente elétrica e a distância da instalação. O sistema calcula a bitola adequada para tensões de 110V e 220V, auxiliando em projetos básicos de instalações elétricas.

## Como executar
1. Abra o Vs Code;
2. Procure a pasta "lib";
3. Dentro da pasta LIB clique em "main.dart";
4. Na barra embaixo procure por "No Device" ou "Windows *windows-x64";
5. Ao encontrar altere para "Chrome (web.javascript)";
6. Após alterar inicie o projeto no "Start Debugging" ou Play;
7. Se tiver realizado corretamente o projeto abrirá no chrome;
